Formatowanie kodu w HTML - Python i Bash

Ta strona demonstruje, jak prawidłowo wyświetlać kod źródłowy na stronach HTML z wykorzystaniem kolorowania składni.

1. Podstawowe formatowanie kodu

Najprostszym sposobem wyświetlania kodu w HTML jest użycie elementów <pre> i <code>:

<pre><code>
def hello_world():
    print("Hello, World!")

hello_world()
</code></pre>

Rezultat (bez kolorowania składni):


def hello_world():
    print("Hello, World!")

hello_world()

Uwaga: Element <pre> zachowuje białe znaki (spacje, tabulatory, nowe linie), co jest kluczowe dla wyświetlania kodu. Element <code> określa, że zawartość jest kodem.

2. Kolorowanie składni z Prism.js

Przykłady kodu Python

Przykład 1: Prosty program

def greet(name):
    """Funkcja wyświetlająca powitanie"""
    return f"Witaj, {name}!"

# Wywołanie funkcji
message = greet("Jan")
print(message)  # Wyświetli: Witaj, Jan!

Przykład 2: Klasa w Pythonie

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
        
    def introduce(self):
        return f"Nazywam się {self.name} i mam {self.age} lat."
        
    @property
    def is_adult(self):
        return self.age >= 18

# Tworzenie obiektu
person = Person("Anna", 25)
print(person.introduce())
print(f"Czy osoba jest dorosła? {person.is_adult}")

#!/bin/bash

# Iteracja przez pliki w katalogu
echo "Szukanie plików .txt..."
count=0

for file in *.txt; do
    if [ -f "$file" ]; then
        echo "Znaleziono: $file"
        count=$((count + 1))
    fi
done

if [ $count -eq 0 ]; then
    echo "Nie znaleziono plików .txt"
else
    echo "Znaleziono $count plików .txt"
fi

Przykład 3: Funkcje i argumenty

#!/bin/bash

# Funkcja do sprawdzania czy plik istnieje
check_file() {
    local file_path="$1"
    
    if [ -f "$file_path" ]; then
        echo "Plik $file_path istnieje."
        echo "Zawartość pliku:"
        cat "$file_path"
        return 0
    else
        echo "Błąd: Plik $file_path nie istnieje!" >&2
        return 1
    fi
}

# Sprawdzanie argumentów
if [ $# -eq 0 ]; then
    echo "Użycie: $0 ścieżka_do_pliku" >&2
    exit 1
fi

# Wywołanie funkcji
check_file "$1"
exit_code=$?

echo "Skrypt zakończony z kodem: $exit_code"

3. Implementacja kolorowania składni

Aby zaimplementować kolorowanie składni na swojej stronie:

  1. Dodaj bibliotekę Prism.js do swojej strony HTML:
<!-- W sekcji <head> -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/prism/1.24.1/themes/prism-okaidia.min.css">
<!-- Na końcu <body> -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.24.1/components/prism-core.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/1.24.1/plugins/autoloader/prism-autoloader.min.js"></script>
  1. Oznacz swoje bloki kodu odpowiednimi klasami języka:
<pre><code class="language-python">
def hello_world():
    print("Hello, World!")
</code></pre>

<pre><code class="language-bash">
#!/bin/bash
echo "Hello, World!"
</code></pre>

Ważne: Pamiętaj o prawidłowym escapowaniu znaków specjalnych HTML, takich jak <, > i &, jeśli dodajesz kod ręcznie. Alternatywnie, możesz użyć narzędzi do automatycznego escapowania.

4. Inne popularne biblioteki do kolorowania składni

Oprócz Prism.js, warto rozważyć:

  • highlight.js - podobna do Prism.js, z szerokim wsparciem dla języków
  • CodeMirror - zaawansowany edytor kodu z kolorowaniem składni
  • Pygments (z backendem serwera) - potężne narzędzie do generowania kolorowanego kodu HTML
